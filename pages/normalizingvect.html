Normalizing a Vector

When we normalize a vector, we make its magnitude 1, but keep its direction the same. We also refer to normalized vectors as unit vectors. In game development, these are used to keep track of directions. This can come in handy when determining normals, which are lines that run perpendicular to another line or surface. Normals can tell you which direction a face on a mesh is facing.
Steps:
First, find the length of the vector using the distance formula. You can apply the distance formula for as many dimensions as you have (you are probably used to seeing just two dimensions when working with triangles). \begin{equation*} \overset{\rightharpoonup}v = \begin{bmatrix} a\\ b\\ c\\ d \end{bmatrix} \xrightarrow[]{} \sqrt{a^2 + b^2 + c^2 + d^2} = ||\overset{\rightharpoonup}v|| \end{equation*} Once you have the length, divide each element in the vector by that length, and it should look something like this: \begin{equation*} \hat{v} = \frac{<a, b, c, d>}{||\overset{\rightharpoonup}v||} \space\space\space\space or\space\space\space\space \hat{v} = \begin{bmatrix}\frac{a}{||\overset{\rightharpoonup}v||}\\ \frac{b}{||\overset{\rightharpoonup}v||}\\ \frac{c}{||\overset{\rightharpoonup}v||}\\ \frac{d}{||\overset{\rightharpoonup}v||} \end{bmatrix} \end{equation*}
Examples:
1) Suppose we have our vector:
\begin{equation*} \overset{\rightharpoonup}A = \begin{bmatrix} 3\\ 4 \end{bmatrix} \end{equation*}
First, find the length using the distance formula:
\begin{equation*} ||\overset{\rightharpoonup}A|| = \sqrt{3^2 + 4^2} = \sqrt{25} = 5 \end{equation*}
Now divide each element by that length:
\begin{equation*} \hat{A} = \frac{<3, 4>}{5} = \begin{bmatrix}\frac{3}{5}\\ \frac{4}{5}\end{bmatrix} \end{equation*}
And we are done!

2) Let's try another one, but this time with more dimensions:
\begin{equation*} \overset{\rightharpoonup}B = \begin{bmatrix} 5\\ 7\\ 3\\ 4\\ 3\\ 6 \end{bmatrix} \end{equation*}
We use the distance formula on the whole thing:
\begin{equation*} ||\overset{\rightharpoonup}B|| = \sqrt{5^2 + 7^2 + 3^2 + 4^2 + 3^2 + 6^2} = \sqrt{144} = 12 \end{equation*}
Now divide each element by that length:
\begin{equation*} \hat{B} = \frac{<5, 7, 3, 4, 3, 6>}{12} = \begin{bmatrix}\frac{5}{12}\\ \frac{7}{12}\\ \frac{1}{4}\\ \frac{1}{3}\\ \frac{1}{4}\\ \frac{1}{2}\end{bmatrix} \end{equation*}
And there you have it!
