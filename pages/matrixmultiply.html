Matrix Multiplication

Unlike Addition and subtraction, multiplcation of matrices is where things get weird. It gets done in a different way than what you might be expecing. We can no longer just apply an operator on two corresponding elements in two matrices. Instead, the row of the first matrix is multiplied by the column of the other, and then added.
Steps:
Subtract the first element in both matrices, then the second, thrid, and so on...
\begin{equation*} \begin{bmatrix} a&b&c\\ d&e&f\\ g&h&i \end{bmatrix} - \begin{bmatrix} j&k&l\\ m&n&o\\ p&q&r \end{bmatrix} = \begin{bmatrix} a-j&b-k&c-l\\ d-m&e-n&f-o\\ g-p&h-q&i-r \end{bmatrix} \end{equation*}
Examples:
1) Here, we have two 3x3 matrices. We subtract each corresponding element:
\begin{equation*} \begin{bmatrix} 1&5&0\\ 6&2&8\\ 1&3&4 \end{bmatrix} - \begin{bmatrix} 9&0&8\\ 3&1&7\\ 5&3&4 \end{bmatrix} = \begin{bmatrix} 1-9&5-0&0-8\\ 6-3&2-1&0-7\\ 1-5&3-3&4-4 \end{bmatrix} = \begin{bmatrix} -8&5&-8\\ 3&1&-7\\ -4&0&0 \end{bmatrix} \end{equation*}
...Which is the difference between the two.

2) In this example, we are dealing with different dimensions:
\begin{equation*} \begin{bmatrix} 12&3&16\\ 1&19&32 \end{bmatrix} - \begin{bmatrix} 14&60&5\\ 42&7&19 \end{bmatrix} = \begin{bmatrix} 12-14&3-60&16-5\\ 1-42&19-7&32-19 \end{bmatrix} = \begin{bmatrix} -2&-57&11\\ -41&12&13 \end{bmatrix} \end{equation*}
And there's your answer.

//The matrices to be added together: var a = [[8, 3, 6], [6, 9, 5], [8, 0, 7], [5, 4, 2]]; //We'll make a 3x4 array var b = [[4, 2, 8], [2, 0, 2], [3, 1, 3], [6, 8, 1]]; var diff = []; for(i = 0; i < a.length; i++) { //Make a fresh row to append to the final matrix: var row = []; for(j = 0; j < a[i].length; j++) { //Add the difference of the two corresponding elements to that row: row.push(a[i][j] - b[i][j]); } //Append the row to the diff matrix: diff.push(row); } //And we're done!----------- //Let's print it out: var output = "difference = \n["; for(i = 0; i < diff.length; i++) { for(j = 0; j < diff[i].length; j++) { //Append the element to the output text: output += " " + diff[i][j] + " "; } //Append a newline character to the output string, //but not if it's the last line: if(i + 1 < diff.length) { output += "\n"; } } //Add the closing bracket: output += "]"; //Finally, print it out: alert(output);
